## How the site works

The site is built with [Jekyll](https://jekyllrb.com/) on GitHub Pages using the Slate theme.

- `index.html` — home page; renders the post and project cards with Liquid
- `_data/posts.yml` — blog posts / analysis reports shown under **Recent Posts**
- `_data/projects.yml` — repositories shown under **Projects**
- `assets/css/custom.css` — card and pagination styling
- `assets/js/pagination.js` — client-side pagination for the cards

To add a new post or project, append an entry to the matching file in `_data/` — no HTML changes needed.
